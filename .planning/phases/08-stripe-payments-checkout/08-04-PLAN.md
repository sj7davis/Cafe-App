| admin browser → platformAdmin.platformFees | Platform admin JWT (env.platformAdminSecret) required — must not accept an owner token |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-10 | Information | payout balance exposure | mitigate | getPayoutBalance scopes to the caller's own franchiseeAccounts.stripeConnectAccountId via owner JWT venueId; no cross-venue balance access. |
| T-08-SC | Tampering | npm installs | accept | No new packages this phase. |
